AI TicTacToe with GUI
{% comment %} Abstract {% endcomment %}
This project, one of my first solo ventures, delves into computer science and AI. It combines two main parts: AI logic and a user-friendly interface made with Java's Swing library.

The AI logic uses pattern detection to decide its moves, analyzing the game board to spot winning or blocking patterns. If it finds one, it plays strategically; otherwise, it makes random moves.

The interface displays a grid for TicTacToe, letting users click to make moves. It also has buttons to choose X or O symbols and shows real-time game progress.

This project not only offers a fun TicTacToe game but also sparked my interest in AI and led me to pursue computer science. It marks the beginning of my journey in this field.
