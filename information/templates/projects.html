AI TicTacToe with GUI
{% comment %} Abstract {% endcomment %}
This project, one of my earliest personal endeavors, showcases my initial exploration into the realms of computer science and artificial intelligence (AI). Consisting of two key components, it combines AI logic and a graphical user interface (GUI) implemented using Java's Swing library.

The AI Logic module employs hard-coded pattern detection algorithms to determine the computer player's moves. It analyzes the game board, utilizing the makeEducatedMove method to identify winning or blocking patterns. When identified, the AI makes strategic moves accordingly; otherwise, it resorts to random moves.

The GUI Implementation features a grid layout representing the TicTacToe board, enabling users to make moves with a simple click. Additionally, it includes player symbol selection buttons (X or O) and provides real-time feedback on game progress.

This project not only provided an interactive TicTacToe implementation but also sparked my interest in AI and machine learning, motivating me to pursue a degree in computer science. It stands as a testament to my early exploration and aspirations within the field.
